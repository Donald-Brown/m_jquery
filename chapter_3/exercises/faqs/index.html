JavaScript FAQs

What is JavaScript

JavaScript is programming language built into the major web browsers. It makes web pages more responsive and saves on round trips to the server.

What is jQuery

jQuery is a library of the JavaScript functions that your most likely to need as you develop websites.

Why is jQuery becoming so popular?

Three reasons

  • It's free
  • It lets you get more done in less time
  • All of it's functions are cross-browser compatible
