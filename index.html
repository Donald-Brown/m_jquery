by: Donald Brown

jQuery Basics

  • jQuery Selectors

    Start by coding the dollar sign ($) to reference the jQuery library. Then in parenthesis code the css selector(s) in quotation marks. The selectors can be elements, ids, classes. The following are how to select elements by relationship.

      Selecting by relationship

    • Descendants: All <p> elements that are decendants of the element with the id notes. 

      $('#notes p')

    • Adjacent siblings: all div elements that are adjacent siblings of the h2 elements.

      $('h2 div')

    • General siblings: all <p> elements that are siblings of ul elements.

      $('ul ~ p')

    • Children: all ul elements that are children of div elements.

      $('div > ul')
      Coding multiple Selectors 
    • $('#notes li, div p')
    • $('p + ul, div ~ p')

  • Calling jQuery methods

    $('selector').methodName(parameters)[.methodName(parameters)]...
      Common jQuery methods
    • val() gets value of form control
    • val(value) sets value of form control
    • text() gets the text of an element
    • text(value) sets the text of an element
    • next([type]) get the next sibling of an element or the next sibling of a specified type if coded
    • submit() submit the selected form
    • focus() move the focus to the selected form control or link

  • Using jQuery event methods

      Two ways to code the ready event
    • Long form: 
      
                  $(document).ready(function() {
                    alert('the DOM is ready');
                  }); // end of ready event
    • Short form: 
      
                  $(function() {
                    alert('the DOM is ready') // (document).ready is assumed
                  }); // end of ready event
    • A click event for all h2's in the ready event 
      
                  $(document).ready(function() {
                    $('h2').click(function() {
                      alert('This heading has been clicked');
                    });    // end of click event handler
                  });      // end of ready event handler

                  // or
                  $(function() {
                    $('h2').click(function() {
                      alert('This heading has been clicked');
                    });    // end of click event handler
                  });
