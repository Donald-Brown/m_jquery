jQuery Basics

  • Using jQuery Selectors

    Start by coding the dollar sign ($) to reference the jQuery library. Then in parenthesis code the css selector(s) in quotation marks. The selectors can be elements, ids, classes. The following are how to select elements by relationship.

      Selecting by relationship

    • Descendants: All <p> elements that are decendants of the element with the id notes. 

      $('#notes p')

    • Adjacent siblings: all div elements that are adjacent siblings of the h2 elements.

      $('h2 + div')

    • General siblings: all <p> elements that are siblings of ul elements.

      $('ul ~ p')

    • Children: all ul elements that are children of div elements.

      $('div > ul')
      Coding multiple Selectors 
    • $('#notes li, div p')
    • $('p + ul, div ~ p')

  • Calling jQuery methods

    $('selector').methodName(parameters)[.methodName(parameters)]...

      Common jQuery methods

    • val() gets value of form control
    • val(value) sets value of form control
    • text() gets the text of an element
    • text(value) sets the text of an element
    • next([type]) get the next sibling of an element or the next sibling of a specified type if coded
    • submit() submit the selected form
    • focus() move the focus to the selected form control or link

  • Some of the most useful jQuery selectors

    • [attribute] - All elements with the named attribute
    • [attribute=value] - All elements with the named attribute and value
    • :contains(text) - All elements that contain the specified text
    • :empty - All elements with no children including text nodes
    • :eq(n) - The element with the index n within the selected set
    • :even - All elements with an even index within the selected set
    • :first - The first element within the selected set
    • :first-child - All elements that are first children of their parent elements
    • :gt(n) - All elements within the selected set that have an index greater than n
    • :has(selector) - All elements that contain the element specified by the selector
    • :header - All elements that are headers (h1,h2,...)
    • :hidden - All elements that are hidden
    • :last - The last element within the selected set
    • :last-child - All elements that are last children of their parent elements
    • :lt(n) - All elements within the selected set that have an index less than n
    • :not(selector) - All elements that are not selected by the selector
    • :nth-child - All elements that are the nth children of their parent elements
    • :odd - All elements with an odd index within the selected set
    • :only-child - All elements that are only children of their parent elemens
    • :parent - All elements that are parents of other elements, including text nodes
    • :text - All elements with the type attribute set to text
    • :visible - All elements that are visible
    • not included - six attribute that let you select attributes with attibute values that contain specific substrings

    Examples

    
$('li:first-child') // li elements that are first children of parent elements
$('table > tr:even') // even tr elements of a table 
$('#notes p:eq(2)') // third decendant p element of an element
$(':text') // all inputs with text as the type attribute

  • Some of the most useful jQuery methods

    • next([selector]) - get the first sibling of each selected element or the first sibling of a specific type if param is coded
    • prev([selector]) - get the previous sibling of each selected element or the prevous sibling of a specific type if param is coded
    • attr(attributeName) - get the value of a specified attribute from the first selected element

  • Some of the most useful event methods

    • ready(handler) - The handler runs when the DOM is ready
    • unload(handler) - the handler runs when the user closes the browser window
    • error(handler) - the handler runs when a JavaScript error occurs
    • click(handler) - the handler runs when the selected element is clicked
    • dblclick(handler) - the handler runs when the selected element is double-clicked
    • mouseenter(handleer) - the handler runs when the mouse enters the selected element
    • mouseover(handler) - the handler runs when the mouse pointer moves over the selected element
    • mouseout(handler) - the handler runs when the mouse pointer moves out of the selected element
    • hover(handlerIn, handlerOut) - the first handler runs when the mouse pointer moves into an element, the second runs when the mouse pointer moves out
    • preventDefault() - stops the defaut event action from happening

    Examples

    
// a handler for all text boxes that clears the double-clicked box
$(':text').dblclick(function() {
$(this).val('');
}) // ends dblclick

// a handler for each img element in a list
$('#image_list img').hover(
function() {
  alert('The mouse pointer has moved into an img element');
},
function() {
  alert('The mouse pointer has moved out of an img element');
}
); // end hover

// preventDefault method that stops the default action of an event
$('#faqs a').click(function(e) {
e.preventDefault();
}); // end click

  • Other event methods to be aware of

    • on(events, handler) - attach an event handler to one or more events
    • off(events, [handler]) - remove an event hanlder from one or more events
    • one(event, handler) - attach an event handler and remove it after it runs one time
    • trigger(event) - trigger the event for the selected element
    • bind(event, handler) - attach an event handler to an event
    • unbind(event, handler) - remove an event handler from an event

    Examples

    
// attach an event handler to an event
$('#clear').on('click', function() {...});
$('#clear').click(function() {...});  // shortcut

// attach handler to two different events
$('#image_list img').on('click mouseover', function() {...}); // same element
let clearClick = function() {...}   // two different elements
$('#clear').click(clearClick);       
$(':text').dblclick(clearClick);

//remove an event handler
$('#clear').off('click');

// an event that only runs once
$('#clear').one('click', function() {...});

// triggering an event
$('#clear').trigger('click'); // trigger method
$('#clear').click();  // shortcut

// using the shortcut method to trigger an event from an event handler
$(':text').dblclick(function() {
$('#clear').click(); // triggers the event of the clear button
}); // end dblclick

How to use effects and animations

  • basic methods for jQuery effects

    • show() - display the selected elements from upper left to lower right
    • hide() - hide the selected elements from lower right to upper left
    • toggle() - display or hide the selected elements
    • slideDown() - display the selected elements in a sliding motion
    • slideUp() - hide the selected elements in a sliding motion
    • slideToggle() - display or hide the selected elements with a sliding motion
    • fadeIn() - display the selected elements by fading them into opaque
    • fadeOut() - hide the selected elements by fading them out to transparent
    • fadeToggle() - display or hide the fading them in or out
    • fadeTo() - adjust the opaicty property of the selected elements to the opacity set by the second parameter. With this method, the duration parameter must be specified
    
//* basic syntax for all of the methods except the fadeTo method
//* methodName( [duration][, callback]);
$('#startup_message').fadeOut(5000).slideDown(1000);

//* basic syntax for the fadeTo method
//* fadeTo(duration, opacity[, callback]);
$('#number2').fadeTo(5000, .2).fadeTo(1000, 1); // chaining is the preferred way
$('#number2').fadeTo(5000, .2, 
function() {
  $(this).fadeTo(1000, 1);
}
);

  • How to use animations

    
//* basic syntax for animations
//* animate({properties}[, duration][, callback]);
$('#faqs h1').animate(
{ fontSize: '275%', opacity: 1, left: 0 }, // the property map
2000
); // end animate

// example with callback
$('#faqs h1').animate(
{ fontSize: '275%', opacity: 1, left: '+=175' }, 
2000,
function() {
  $('#faqs h2').next().fadeIn(1000).fadeOut(1000);
}
); // end animate

  • The delay, swap, and finish methods

    • delay(duration) - delay the start of the next animation in queue
    • stop([clearQueue][, jumpToEnd]) - Stop the current animation for the selected element. the two parameters are boolean with false default values. If set to true, the first parameter clears the queue so no additional animations run. The second parameter causes the current animation to be completed immediately
    • finish([queue]) - Stop the animation for the selected element, clear the queue, and complete all animations for the selected elements

  • Using Easings

//* syntax for basic animate method
animate({ properties }[, duration][, easing][, callback]);

Working with forms and controls

jQuery Selectors for Form Controls

  • :input - All input, select, textarea, and button elements
  • :text - All text boxes, input elements with type equal to text
  • :radio - All radio buttons, input elements with type equal to button
  • :checkbox - All check boxes, input elements with type equal to checkbox
  • :file - All file upload fields, input elements with type equal to file
  • :password - All password fields, input elements with type equal to password
  • :submit - All submit buttons and button elements, input elements with type equal to submit and button elements
  • :reset
    • - All reset buttons, input elements with type equal to reset
  • :image - All image buttons, input elements with type equal to image
  • :button - All buttons, input elements with type equal to button
  • :disabled - All disabled elements, elements that have the disabled attribute
  • :enabled - All enabled elements, elements that don't have the disabled attribute
  • :checked - All check boxes and radio buttons that are checked
  • :selected - All options in select elements that are selected

jQuery methods for getting and setting values of controls

  • val() - Gets the value of a text box or other form control element
  • val(value) - Sets the value of a text box or other form control element

jQuery method for trimming an entry

  • trim() - Removes all spaces at the start and end of a string

jQuery event methods for forms

  • focus(handler) - The handler runs when the focus moves to the selected element
  • blur(handler) - The handler runs when the focus leaves the selected element
  • change(handler) - The handler runs when the value in the selected element is changed
  • select(handler) - The handler runs when the user selects text in a text or textarea box
  • submit(handler) - The handler rus when a submit button is clicked

jQuery methods that trigger events

  • focus() - Moves the focus to the selected element and triggers the focus event
  • blur() - Removes the focus from the selected element and triggers the blur event
  • change() - Triggers the change event
  • select() - Triggers the select event
  • submit() - Triggers the submit event for a form

DOM manipulation and traversal methods

Methods for working with attributes

  • attr(name) - gets the value of the named attribute from the first selected element
  • attr(name, value) - sets the named attribute for each selected element
  • attr(map) - sets the value of multiple attributes specified by the name/value pairs in the attribute map
    ex. $("#image").attr( {"src": "book1.jpg", "alt": "Book 1"} );
  • attr(name, function) - sets the value of the named attribute to the value returned by the function
  • removeAttr(name) - removes the named attribute from each selected element

$('#image').attr('src', 'book.jpg');

// Use map to set two values
$('#image').attr( {'src': 'book.jpg', 'alt': 'Book 1'} );

// Use a function to set href attribute
$('aside a').attr('href', function(index) {
  let href = '#heading' + (index + 1);
  return href;
});

// remove id attribute
$('#faqs h2').removeAttr('id');

Methods for working with class attributes

  • hasClass(name) - returns true if the named class is present in any selected elements
  • addClass(name) - adds the named class to each selected element
  • addClass(function) - adds the class specified by the value returned by the function
  • removeClass(name) - removes the named class from each of the selected elements
  • removeClass(function) - removes the class specified by the value returned by the function
  • toggleClass(name) - if the named class is present, remove it from each selected element. otherwise, add it
  • toggleClass(function) - if the class specified by the value returned by the function is present, remove it. Otherwise, add it

// test if element has a specified class
if ($('#faqs').hasClass('closed')) {
  do something;
}

// add a class
$('#faqs h2').addClass('minus');

// remove a class
$('#faqs h2').removeClass('minus');

Methods for DOM replacement

  • val() - gets the value of the first selected form element
  • val(value) - sets the value of the selected form elements
  • val(function) - sets the value of the selected form elements to the value returned by the function
  • text() - gets the combined text of all selected elements inclucing their descendants
  • text(textString) - sets the contents of each selected element to the specified text
  • text(function) - sets contents of each selected element to the text returned by the function
  • html() - gets the html contents of the first seclected element
  • html(htmlString) - sets the HTML contents of each selected element to the specified HTML
  • html(function) - set the HTML contents of each selected element to the HTML string returned by the function
  • replaceWith(content) - replaces each selected element with the specified content. this counld be an HTML string, a DOM element, or a jQuery object
  • replaceAll(target) - replaces the target element with the selected element. this is reverse of how the replaceWith method is coded

// display the HTML in the aside 
alert($('aside').html());

// put a h2 into the aside
$('aside').html(`<h2>Table of Contents</h2>`);

// replace all elements with class old with an h2 element
$('.old').replaceWith('<h2></h2>');

// replace all elements with class old with an h2 element
$('<h2></h2>').replaceAll('.old');

Methods for DOM inserting and cloning

  • prepend(content) - Inserts the specified content at the start of each selected element
  • prepend(function) - Inserts the content returned by the function at the start of each specified element
  • prependTo(target) - Inserts all of the selected elements at the start of each of the target elements
  • append(content) - Inserts the specified content at the end of each selected element
  • append(function) - Inserts the content returned by the function at the end of the selected elements
  • appendTo(target) - Inserts all of the selected elements at the end of each target elements
  • before(content) - Inserts the specified content before each selected element
  • before(function) - Inserts the content returned by the function before each of the selected elements
  • insertBefore(target) - Inserts all of the selected elements before each of the target elements
  • after(content) - Inserts the specified content after each of the selected elements
  • after(function) - Inserts the content returned by the function after each of the selected elements
  • insertAfter(target) - Inserts all of the selected elements after each target element
  • clone([withEvents]) - Create a copy of the selected elements. The parameter is a Boolean that indicates if the event handlers are also copied

// insert an h2 at the end of an aside
$('aside').append('<h2>Table of Contents</h2>');

// insert an a element after the last p element in an article
$('article p:last').after('<a href="#top">Back to top</a>');

// insert the a elements in an article after the h2 elements in an aside
$('article a').insetAfter($('aside h2'));

// clone the a elements and inset them after the h2 element in an aside
$('article a').clone().insertAfter($('aside h2'));

Methods for DOM wrapping and removal

  • wrap(element) - Wraps the specified element around each selected element
  • wrap(function) - Wraps the element returned by the function around each of the specified elements
  • wrapAll(element) - wraps the specified element around all of the selected elements
  • wrapInner(element) - Wraps the specified element around the content of each selected element
  • wrapInner(function) - Wraps the element returned by the function around the content of each selected element
  • empty() - Removes all of the child nodes of each selected element
  • remove([selector]) - Removes the selected elements from the DOM. If a selector is specified, it filters the selected elements
  • unwrap() - Removes the parent of each selected element

// wrap all a elements in h2 elements
$('a').wrap('<h2></h2>');

// wrap the h1 text in an a element
$('article h1').wrapInner('<a id="top"></a>');

// remove the first a element in an article
$('article a:first').remove();

Methods for working with styles

  • css(name) - Gets the value of the named property from the first selected element
  • css(name, value) - sets the value of the named propety for each of the selected elements
  • css(map) - Sets the value of multiple properties specified by the name/value pair map
  • css(name, function) - Sets the value of the named propety to the value returned by the function
  • height() - Gets the height of the first selected element. This height does not include padding, margins, or borders
  • height(value) - Sets the height of each selected element
  • innerHeight() - Gets the height of the first selected element including padding but not margins or borders
  • outerHeight([includeMargin]) - Gets the height of the first selected element including padding and borders. If the parameter is set to true, it also includes margins.
  • width() - Gets the width of the first selected element. This doesn't include padding, margins, or borders
  • width(value) - Sets the width of each selected element
  • innerWidth() - Gets the width of the first selected element including padding but not margins or borders
  • outerWidth([includeMargin]) - Gets the width of the first selected element including padding and borders. If the parameter is set to true, it also includes margins

// Set all h2 elements color to blue
$('h2').css('color', 'blue');

// use map to set two css properties
$('h2').css( {'color': 'blue', 'font-size': '150%'} );

// get the height of an article element
let height = $('article').height();

Methods for positioning elements

  • offset() - Gets the coordinates of the first selected element and returns an object with top and left properties. These coordinates are relative to the document
  • offset(coordinates) - Sets the coordinates for each of the selected elements relative to the document. The parameter is an object with left and top properties
  • position() - Gets the coordinates of the first selected element and returns an object with top and left properties. These coordinates are relative to the parent
  • scrollTop() - Gets the position of the vertical scroll bar for the first selected element
  • scrollTop(value) - Sets the position of the vertical scroll bar for each selected element
  • scrollLeft() - Gets the position of the horizontal scroll bar for the first selected element
  • scrollLeft(value) - Sets the position of the horizontal scroll bar for each selected element

// get the top offset for an article element
let offsetTop = $('articele').offset().top;

// Set the offset coordinates for an aside element
let asideCoordinates = new Object();
asideCoordinates.top = 200;
asideCoordinates.left = 100;
$('aside').offset(asideCoordinates);

// Move the scroll bar for the window to the offsetTop
$(window).scrollTop(0);

Two event methods used with DOM manipulation

  • on(events[, selector], handler) - Attach an event handler to one or more events of the selected elements
  • off(events[, selector][, handler]) - remove an event handler from one or more events

// a click event method that handles the click event of an h2 element
$('#accordion h2').click(function() {
  // the code for the event handler
});

// an on event method that handles the click event of an h2 element
$('#accordion').on('click', 'h2', function(){
  // the code for the event handler
});

// an off event method that removes the first click event above
$('#accordion h2').off('click');

// an off event method that removes the second click event handler above
$('#accordion').off('click', 'h2');

DOM traversal methods

  • sibling([selector]) - Gets the siblings of each selector, optionally filtered by a selector
  • next([selector]) - Gets the first sibling that follows each selected element, optionally filtered by a selector
  • nextAll([selector]) - Gets all siblings that follow a selected element, optionally filtered by a selector
  • nextUntil(selector) - Gets all siblings that follow a selected element up to but not including the element specified by the selector
  • prev([selector]) - Gets the first sibling that precedes each seleceted element, optionally filtered by the selector
  • prevAll([selector]) - Gets all siblings that precedes each selected element, optionally filtered by a selector
  • prevUntil(selector) - Gets all siblings that precede each selected element, up to but not including the element specified by the selector
  • children([selector]) - Gets the children of each selected elemetnt, optionally filtered by the selector
  • parents([selector]) - Gets the ancestors of the selected elements, optionally filtered by the selector
  • parentsUntil(selector) - Gets the ancestors of each selected element, up to but not including the element specified by the selector
  • parent([selector]) - Gets the parent of each selected element, optionally filtered by the selector
  • offsetParent() - Gets the closest ancestor of an element that is positioned
  • closest(selector[, context]) - Gets the first element that matched the specified selector, beginning at the current element and going up the DOM tree. If the context parameter is specified, it specifies a DOM element within which the selected element must be found
  • find(selector) - Gets the descedants of each selected element after it has been filtered by the specified selector

// get the prev paragragh sibling of an element
let previousParagragh = $('#last_heading').prev('p');

// get the parent of an element
let parent = $('#faqs').parent();

// get the span elements within p elements within an article element
$('article p').find('span').css('color', 'red');

Filtering methods

  • filter(selector) - Reduces the set of selected elements to those that match the selector
  • filter(function) - Reduces the set of selected elements to those that pass the functions test
  • not(selector) - Removes the elements specified by the selector from the set of selected elements
  • not(elements) - Removes the specified elements from the set of selected elements
  • not(function) - Removes the elements that pass the functions test from the set of selected elements
  • has(selector) - Reduces the set of selected elements to those that have a descendant that matched the selector
  • eq(index) - Reduces the set of selected elements to the one that has the specified index
  • first() - Reduces the set of selected elements to the first one in the set
  • last() - Reduces the set of selected elements to the last one in the set
  • slice(start[, end]) - Reduces the set of selected elements to those within the range of the indexes that are specified by the parameters
  • end() - Returns the set of selected elements to its previous state

// change a property for the h2 elements that are in the 'best' class
$('h2').filter('.best').css('color', 'red');

// change a property for all p elements except the ones with the 'first' class
$('p').not('.first').css('text-indent', '1.5em');

// Hide all images in the 'slides' element except for the one with index 0
$('#slides img').slice(1).hide();

// work with images in a set
$('#slides img')
                .first().fadeOut(1000)  // fade out first img element in set
                .next().fadeIn(1000)    // fade in the next element in set
                .end()                  // return to first element in set
                .appendTo('#slides');   // append first element to end of set
